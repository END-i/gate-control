# Release v0.1.3

Date: 2026-04-29
Tag: v0.1.3

## Summary

This release stabilizes the delivery pipeline and local/CI runtime behavior after enabling automated versioning.

## Highlights

- CI now installs backend Python dependencies before running the quality gate.
- Backend startup now guarantees media directory creation before static mount.
- CI actions are configured for Node.js 24 compatibility mode.
- Frontend quality gate flow is stabilized:
  - fixed pnpm workspace configuration in frontend
  - installs frontend dependencies before lint/typecheck/tests in check-all script
- Release/version automation chain remains active:
  - CI -> Versioning -> Release

## Validation

- Full local quality gate passes (`./scripts/check-all.sh`).
- Docker smoke checks verified:
  - backend health
  - frontend reachable
  - auth login
  - webhook flow
  - logs and system status endpoints

## Known Notes

- Host frontend port may be configured via `.env` (`FRONTEND_PORT`) if port 3000 is occupied.
