# Release v0.1.0

Date: 2026-04-29
Branch: main

## Highlights

- FastAPI backend for ANPR gate control with JWT admin auth.
- Vehicle allowlist CRUD with validation and duplicate plate protection.
- ANPR webhook ingestion with token or HMAC verification modes.
- Access decision flow with relay trigger integration.
- Access logs API with filters, pagination, and SSE stream.
- Statistics and system status endpoints.
- SvelteKit admin frontend with login, dashboard, vehicles, logs, image preview, and language switcher.
- Localized UI dictionaries: en, ru, uk, bg.
- Backend and frontend automated tests.
- Quality gate script for lint, typecheck, tests, and optional docker builds.
